# Rosa-dos-ventos_petrolina# 

Análise Interativa de Dados de Vento (Petrolina – 2010)

Projeto de análise de dados meteorológicos com foco em aplicações em energia eólica.

## Descrição

Este projeto realiza o processamento e análise de dados de vento (velocidade e direção) para a região de Petrolina (PE), utilizando dados da rede SONDA/INPE.

Foi desenvolvida uma aplicação em Ruby para análise interativa a fim de construir uma boa visualização dos dados, a partir de uma rosa dos ventos.

## Funcionalidades

- Processamento de dados em CSV
- Análise de séries temporais de vento
- Geração de rosa dos ventos
- Identificação de padrões de direção e intensidade

## Aplicação

O projeto permite avaliar padrões de vento e auxiliar na análise de potencial de geração eólica.

## Estrutura

- `data/` → dados de entrada (exemplo)
- `src/` → código fonte
- `images/` → visualizações

## ▶️ Como executar

1. Instale Ruby 
2. Execute:

ruby src/rosa_dos_ventos_main.rb
